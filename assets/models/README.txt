Place rigged humanoid .glb files here.

Expected file naming convention:
  vessel.glb

  ten_shadows.glb
  hakari.glb
  perfection.glb
  strongest_of_history.glb
  boss_sukuna.glb

  default.glb          (generic fallback model)

Requirements:
  - Format: glTF Binary (.glb)
  - Rigged humanoid skeleton (Mixamo-compatible naming preferred)
  - Recommended bone names: Hips, Spine, Spine1, Spine2, Neck, Head,
    LeftShoulder, LeftArm, LeftForeArm, LeftHand,
    RightShoulder, RightArm, RightForeArm, RightHand,
    LeftUpLeg, LeftLeg, LeftFoot,
    RightUpLeg, RightLeg, RightFoot
  - Optional embedded animations: idle, punch, block, special, awaken
  - Max poly count: ~15k tris recommended for mobile performance

Free sources for rigged humanoid models:
  - Mixamo (mixamo.com) — free rigged characters + animations
  - ReadyPlayerMe (readyplayer.me) — avatar creator with GLB export
  - Sketchfab (sketchfab.com) — search for CC0/free rigged characters
  - Quaternius (quaternius.com) — free low-poly characters
